Perguntas e Respostas:

a) Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?
Resposta: O atributo "placehold" da uma dica do que deve ser inserido para o usuário.

b) Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
Resposta: O atributo "checked" significa que o usuário deve escolher uma das opções apresentadas.

c) Na tag <input> para que serve o atributo "value"?
Resposta: O atributo "value" serve para definir o valor da tag input.

d) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
Respota: O atributo a ser utilizado deve ser o "checked".

e) Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?
Resposta: A tag label associa um texto ao input e o for serve para explicitar esse label ao input.

f) Na tag <option> para que serve o atributo "selected"?
Resposta: A tag "selected" serve para informar uma linha suspensa.

g) Nas tags de formulário para que serve o atributo "required"?
Resposta: O atributo "required" informa para o usuário que esse campo é obrigatório;

h) Para que serve a tag <textarea>?
Reposta: O "textarea" define uma entrade de dados com multiplas linhas.

i) Na tag <form> para que serve o atributo "action"?
Resposta: O action define o que deve ser feito após o submit.

j) Na tag <form> para que serve o atributo "method"?
Resposta: O method define o método de http que será usado após o submit.

k) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?
Reposta: Deve ser usado o method post.

l) Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?
Respota: Deve-se utilizar o method post

m) O que as tags <fieldset> e <legend> produzem? Resposta: Fieldset é utilizado para agrupar elementos e o legend define uma legenda.
